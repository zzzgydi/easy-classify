# easy-classify

一个简单的文本分类工具平台，基于 fasttext 以及 jieba 分词实现。

## 启动项目

分别启动前端开发服务以及后端服务

```bash
# 一个shell跑前端
cd client

npm install

npm run dev

# 一个shell跑后端
python server/app.py
```
